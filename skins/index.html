Usage

Up and running in four steps

01

Download a skin

Grab any skin folder from GitHub. index.html is your starting point — customise it as much as you need.

02

Edit config.js

Set your device address, map your JSON keys, name your commands. Start here before touching anything else.

03

Align your device

Your device just needs to broadcast state as JSON and accept commands in the same shape. That's the full contract.

04

Open in any browser

Phone, tablet, desktop, touchscreen panel. Serve from anywhere — or open the file directly. Works offline.
